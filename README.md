# Email Spam Detection Using Machine Learning 

An end-to-end Machine Learning project that classifies emails into **Spam** or **Ham (Legitimate)**. This project implements Natural Language Processing (NLP) techniques and classification algorithms to accurately detect fraudulent or unwanted emails.

---

##  Features
* **Text Preprocessing:** Cleans raw email text by removing punctuation, stop words, and applying stemming/lemmatization.
* **Feature Extraction:** Converts text data into numerical vectors using techniques like TF-IDF (Term Frequency-Inverse Document Frequency) or CountVectorizer.
* **Machine Learning Models:** Implements classification algorithms (such as Naive Bayes, Logistic Regression, or Support Vector Machines) to train and evaluate the data.
* **Performance Evaluation:** Provides insights into model performance using accuracy scores, confusion matrices, and classification reports.
